Política de Privacidad

Última actualización: 9 de abril de 2025

Este GPT personalizado utiliza la API pública de Crossref para buscar artículos científicos a partir de las palabras clave, autores, fechas y otras opciones que usted proporcione.

🔍 ¿Qué información recopilamos?

Este GPT solicita únicamente los siguientes datos cuando usted hace una consulta:

  • Palabras clave de búsqueda
  • Nombre del autor (opcional)
  • Nombre de la revista (opcional)
  • Rango de fechas (opcional)

🔐 ¿Se almacena su información?

No. Este GPT no almacena, guarda ni comparte ninguna de las consultas realizadas por los usuarios. Toda la información ingresada se utiliza únicamente para generar una solicitud a la API de Crossref en tiempo real.

🌐 ¿Con quién se comparte la información?

La información ingresada por el usuario es enviada a través de una solicitud HTTPS a la API de Crossref (https://api.crossref.org). Esta es una API pública que no requiere autenticación ni identificación personal.

📘 Sobre la API de Crossref

Crossref es un sistema de identificación de publicaciones académicas. Puedes consultar su política de privacidad aquí: https://www.crossref.org/privacy/

📞 Contacto

Si tienes preguntas o inquietudes sobre esta política, puedes contactar al desarrollador de este GPT directamente desde la interfaz de OpenAI.
